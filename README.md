# Airbnb Clone Project

A modern Airbnb homepage clone with additional routes, detailed pages, smooth animations, and a beautiful responsive UI. This project is built to showcase frontend and backend integration using **Next.js, TypeScript, shadcn/ui, Framer Motion, MongoDB, and Express.js**.

[**Live Demo**](https://airbnb-five-xi.vercel.app/) | [**Backend Repository**](https://github.com/Nakib64/airbnb_server)

---

## Features

- **Homepage**: Cloned Airbnb homepage with responsive design.  
- **Dynamic Routes**:
  - Hotels by division
  - Experiences by division
  - Services by category
- **Details Pages**: Each card (hotel, experience, service) has its own details page.  
- **Smooth Animations**: Implemented using **Framer Motion** for hover effects, page transitions, and interactive components.  
- **Beautiful Navbar**: Sticky, responsive navbar with scroll animation and mobile-friendly modal search.  
- **Search Functionality**: Users can search for hotels, experiences, and services with dynamic filters for check-in/out dates and guest count.  
- **Responsive Design**: Fully responsive across mobile, tablet, and desktop.  

---

## Tech Stack

- **Frontend**:  
  - [Next.js](https://nextjs.org/) (React framework)  
  - [TypeScript](https://www.typescriptlang.org/)  
  - [shadcn/ui](https://ui.shadcn.com/) for UI components  
  - [Framer Motion](https://www.framer.com/motion/) for animations  
- **Backend**:  
  - [Express.js](https://expressjs.com/)  
  - [MongoDB](https://www.mongodb.com/) for data storage  
- **Deployment**: Vercel  

---
## Usage

- **Homepage**: Explore featured listings, hotels, experiences, and services.  
- **Search**: Use the search bar to filter by destination, dates, and guest count (or services).  
- **Details Page**: Click on any card to view detailed information.  
- **Responsive Navigation**: Works on mobile with modal search and smooth animations.
